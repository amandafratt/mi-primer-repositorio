Device pixel ratio (DPR)

Tipues de píxels

Existeixen dos tipus de píxels:

  • Píxels físics: són els píxels reals que es troben en un monitor o un mòbil. Quan ens referim a una pantalla de 1920x1080 sempre estem parlant de píxels físics.
  • Píxels CSS o del viewport: és la unitat de mesura amb la que treballa CSS per pintar elements en el viewport, no sempre són del mateix tamany que un píxel físic.

Explicació

Device Pixel Ratio es pot abreviar com DPR. El DPR és el ratio entre els píxels físics i els píxels CSS o del viewport. Aquest seria un exemple: si el meu DPR és de 2 això vol dir que 1 pixel CSS serà de 4 pixels físics (2 píxels físics d'altura x 2 píxels físics d'amplada)

Webs responsives

Una web responsive és una web que s'adapta a qualsevol dispositiu: mòbils, televisons, tablets, etc.
