Inputs amb text

ANAR AL ÍNDEX

És el tipus d'input per defecte. Si no poseu cap atribut type en un input, tindreu un input de tipus text.

Atributs habituals

  • Atribut name

    • Name
    Surname

  • Atribut maxlenght

    • Per limitar el nombre de caràcters que un usuari pot introduir en un camp de text (en aquest cas 7).

  • Atribut size

    • Per indicar l'amplasa de l'entrada del text. Aquest atribut no s'hauria d'utiltzar.

  • Atribut value

    • Per establir un valor predeterminat en un camp d'entrada de text HTML. Quan es carrega una pàgina web que conté un atrbut value definit, el valor especificat es mostrarà automàticament en el camp.

  • Placeholder

    • Guiar a l'usuari perque posin allò que desitgem. Es pinten amb un color més fluixet i desapareix quan posem la nostra resposta

  • Pattern

    • Aquest atribut va acompanyat d'una expressió regular o regex. Immpedeix posar qualsevol text que no sigui el que hem posat en l'expressió regular.

  • Required

    • Aquest atribut fa que la casella s'hagi d'omplir obligatoriament.
