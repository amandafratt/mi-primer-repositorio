Input de tipus radio

ANAR AL ÍNDEX

Un input de tipus radio ens permet construir caselles de selcció múltiple.

Atributs importants

Necessitem uns atributs específics per arreglar el problema precedent.

Atribut name

Hem d'asscociar les opcions amb l'atribut name amb e mateix valor a totes les opcions.

Atribut value

Necessitem donar un valor a cada opció perquè el servidor l'enregistri

Atributs accessoris booleans

Hi ha alguns atributs booleans que ens poden anar bé en certes ocasions

  • checked
  • disabled
  • autofocus
  • required

Atribut checked

Et selecciona una opció

Atribut disabled

Prohibeix una opció

Atribut autofocus

Suggereix una opció

Atribut required

Obliga a seleccionar una opció
