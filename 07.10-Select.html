Element html select bàsic

Serveix per seleccionar entre una llista d'opcions un o més items.

Si es posa l'etiqueta select sola, activa una llista deplegable però no es veu res.

Etiqueta option

Afageix un item per poder triar en el menú desplegable. El select porta l'atribut name i les etiquetes option els atributs value.

Atribut múltiple

L'atribut múltiple, que es posa dins l'etiqueta select, ens permet escollir-ne més d'un si apreten la tecla Control
