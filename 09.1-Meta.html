Meta

ANAR AL ÍNDEX

En l'etiqueta head ja hem vist l'etiqueta title i la única etiqueta que coneixem és la meta charset. Feu servir emmet per escriure les etiquetes. D'altres les importareu.

Etiqueta meta charset

Serveix per la codificació dels caràcters de la pàgina web.

<meta charset= "UTF-8">

Etiqueta meta name viewport

Serveix pero treballar amb dissenys que s'han de adpatar als mòbils. La majoria d'etiquetes meta tenen un name i un content. El name fa referència quin tipus de metadades li estem donant al navegador i quin és el contingut d'aquella metada.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Etiqueta meta name description

La metadada description porta un text com a contingut. Aquest text és el que apareix sota el link d'una pagina web quan cerques a google.

<meta name="description" content= "The MDN...>

Etiqueta meta name author

La metadada author porta un text com a contingut que explica qui és el propietari de la web o qui l'ha construit
